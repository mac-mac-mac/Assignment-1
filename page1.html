CPSC 2125 - ASSIGNMENT 1

Concepts Covered in CPSC 2125:

  1. CSS Box Model
    1. Every HTML content element is contained in a rectangular box
    2. Each element has a a border surrounding its content, which may or may not be visible
    3. Each element may also have extra inner space between the content and border (padding) or extra space outside the border between different elements (margin)
  2. Absolute vs Relative Paths
    1. Every file has a path which gives information about how to find where that file is actually located
    2. Absolute paths contain the full list of directories or folders required to locate a particular file starting at the top-level root directory
    3. Relative paths are shortened versions of paths which assume that you will continue your path from the current active directory, rather than starting at the top-level root directory
  3. Grid Layout
    1. The default layout for a webpage (when no additional styling is applied) is to display elements one after another, straight down the page in the order they are listed within the HTML
    2. The CSS Grid Layout is a way to easily organize elements into rows and columns
    3. One common method of using CSS grids is to label each important element with class names, then explicitly define a template of where each section should display using grid-template-areas
